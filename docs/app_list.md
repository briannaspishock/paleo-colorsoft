# research suite
*apps are selected for Fire OS 5 (Android 5.1) compatibility.*

### core interface
- **Nova Launcher** (v1.15.20) - minimalist, distraction-free
- **Whicons** - applied via Nova/Niagara settings only

### reading and markup
- **Libby** (v6.4.0) - can use Silk browser if legacy version doesn't load
- **Foxit PDF Lite** (v3.8.0) - for marking up academic papers

### optical layer
- **CF.lumen** (v3.70) - for the R-G-B recipe.
- **Red Moon** (v3.8) - secondary saturation damper
